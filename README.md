No. The software saves all files on your local hard drive. Your research stays private to your computer.

**Can I delete items from the wiki?**
Yes. Right-click any file in the list and select delete. The system removes the information from the knowledge base immediately.

**What happens if the internet cuts out?**
The software runs locally. You can browse your saved wiki entries without an internet connection. You only need a connection when adding new links or PDFs.

**How much storage will this take?**
The software footprint is small. However, storing many large PDF files will eventually use significant disk space. Clean your wiki occasionally if you notice your drive filling up.
